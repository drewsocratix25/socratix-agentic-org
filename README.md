# The Agentic Org — A Socratix Interactive Journey

A single-file, offline, facilitator-driven web experience for a ~60-minute Vistage-style workshop on AI transformation, built around Socratix's three foundations: **Business Strategy · People & Culture · Technology**.

## What it is
A narrative, choose-your-own-adventure prototype. Your guide **Socrates** runs a short spoken assessment, then the room explores AI strategy through three Greek characters — **Jason** (The Navigator), **Apollo** (The Visionary), and **Atlas** (The Pragmatist) — making Buy / Build / Ignore calls that ladder up to a take-home roadmap and the Socratix AI Readiness Sprint.

## Run it
Just open `index.html` in any modern browser (Chrome, Safari, Edge). No install, no build step, no internet required for the core experience (video embeds are the only optional network use). Room progress is saved in your browser.

**Facilitator controls:** press **F** for the timer, chapter jump, reveal toggle, discussion timers, and reset.

## Host it (optional)
Because it's a single `index.html`, you can serve it free with **GitHub Pages**:
1. Push this repo to GitHub (see below).
2. In the repo, go to **Settings → Pages → Build and deployment → Source: Deploy from a branch**, pick `main` / root.
3. Your workshop tool is live at `https://<your-username>.github.io/<repo-name>/`.
